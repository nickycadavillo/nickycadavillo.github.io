Rider is an AngularJS engineered and mobile-first designed website that allows users to compare the current rates and ETAs of transportation options; from local taxis, ridesharing companies uber and lyft, to any available public transportation.

Each option is integrated and deep linked for easy access; Users can call a local taxi from the app, go directly to the ridesharing app they've chosen, and get transit directions from Google Maps right from Rider.

  • API integration

Chat App is a jQuery and bootstrap chat assistant. Using the ampersand, Chat App is able to reply to run a command. The application calls from several APIs to gather weather, temperature, giphys, movie information, calcualtions. The user can also customize the background color of the app.
